# Sesión 1: Fundamentos y Adquisición de Datos

**Duración**: 5 horas

## Objetivos

- Comprender la arquitectura de recolección de datos dual-domain
- Extraer datos crudos desde Elasticsearch
- Explorar y validar la estructura de los datos JSONL
- Transformar datos JSONL en datasets CSV estructurados
- Evaluar la calidad del CSV resultante
